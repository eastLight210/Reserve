# Reserve

서울대 풋살장 예약

* git clone 혹은 다운로드 이후 
"pip install -r requirements.txt"
를 이용해 필요한 패키지를 설치해주세요.  

* Account.txt, Member.txt로 계정 정보와 멤버를 작성하여 폴더에 넣어주세요

* 이후 날짜, 시간 등을 수정하여 사용하면 됩니다.

# Account.txt 예시  
myid  
mypasswd
